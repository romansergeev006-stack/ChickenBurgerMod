# Chicken Burger Mod

A Minecraft Forge mod for version 1.16.5 that adds a delicious chicken burger with a magical transformation effect!

## Features

- 🍗 **Chicken Burger** - A new food item with 10 hunger points and high saturation
- ✨ **Chicken Transformation** - Eating the burger transforms you into a chicken for 60 seconds with:
  - Speed effect (Speed III)
  - Jump boost (Jump Boost III)
  - Slow falling effect
  - Reduced hitbox (chicken size)

## Installation

1. Download Forge 1.16.5 from [minecraftforge.net](https://minecraftforge.net)
2. Install Forge for 1.16.5
3. Place the mod jar in your `mods` folder
4. Launch Minecraft with the Forge profile

## Building

```bash
./gradlew build
```

The mod jar will be in `build/libs/`

## Running in Dev

```bash
./gradlew runClient
```
